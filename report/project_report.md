# Abstract
This project focuses on scalable analysis and predictive modeling on a large-scale e-commerce transaction dataset. The dataset contains more than 19 million rows and was processed using both Pandas and PySpark. Multiple machine learning approaches were tested and compared. A Django dashboard was developed to visualize the results and allow interactive purchase prediction.

---

# 1. Introduction
E-commerce platforms generate huge amounts of transaction data. Traditional tools often fail to process such large datasets efficiently. Therefore, this project uses Big Data tools such as PySpark along with machine learning models to analyze customer behavior and predict purchases.

---

# 2. Objectives
- Perform scalable analysis on large e-commerce data
- Compare Pandas and PySpark
- Build multiple ML models
- Detect and remove data leakage
- Create a dashboard for visualization and prediction
