# Scalable Analysis and Predictive Modeling on Large-Scale E-commerce Data

## Project Overview
This project focuses on scalable data processing and predictive modeling on a large-scale e-commerce transaction dataset.

The project simulates a real-world Big Data pipeline using:
- Pandas
- PySpark
- Machine Learning
- Django Dashboard

The dataset contains millions of e-commerce transaction records and is used to:
- Analyze customer behavior
- Compare ML approaches
- Predict whether a user is likely to purchase
